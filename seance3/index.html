Git - the ultimate VCS

Basic commands

git init

  • Initiate empty repository
  • Use at the top of the folder hierarchy to version
jopasserat@vapula:/tmp$ cd seance3/exo1/
jopasserat@vapula:/tmp/seance3/exo1$ git init
Initialized empty Git repository in /private/tmp/seance3/exo1/.git/

git add

  • Start tracking a file
  • Takes files or whole folders as parameter
jopasserat@vapula:/tmp/seance3/exo1$ ls
bar	foo	toto
jopasserat@vapula:/tmp/seance3/exo1$ git add bar
jopasserat@vapula:/tmp/seance3/exo1$ git add .

git status

  • Display the current status of the working tree (what is to be commited, what is staged for commit...)
jopasserat@vapula:/tmp/seance3/exo1$ git status
# On branch master
#
# Initial commit
#
# Changes to be committed:
#   (use "git rm --cached ..." to unstage)
#
#	new file:   bar
#	new file:   foo
#	new file:   toto
#

git commit

  • One of the most important command!
  • Creates a snapshot of each file staged for commit
jopasserat@vapula:/tmp/seance3/exo1$ git commit bar -m 'First commit of bar file'
[master (root-commit) 6ba7d15] First commit of bar file
0 files changed
create mode 100644 bar
jopasserat@vapula:/tmp/seance3/exo1$ git commit -am 'Commit the remaining files'
[master cd2c39c] Commit the remaining files
0 files changed
create mode 100644 foo
create mode 100644 toto

git log

  • Browse the commits history
  • Displayed using /usr/bin/less on UNIX (same commands as Vi...)
jopasserat@vapula:/tmp/seance3/exo1$ git log
commit cd2c39cc6a6bb6a08f3d74c37ecc0dfd71e5d604
Author: jH@CKtheRipper 
Date:   Tue Mar 12 22:48:32 2013 +0100
    
    Commit the remaining files
    
commit 6ba7d15d3fe75bfe279f51eae81cdb11016afee1
Author: jH@CKtheRipper 
Date:   Tue Mar 12 22:48:03 2013 +0100
        
    First commit of bar file

git diff

  • Show the changes between the working tree version of a file and a commited version
  • + for inserted lines
  • - for removed lines
jopasserat@vapula:/tmp/seance3/exo1$ vi bar
jopasserat@vapula:/tmp/seance3/exo1$ git diff bar
diff --git a/bar b/bar
index e69de29..270c611 100644
--- a/bar
+++ b/bar
@@ -0,0 +1 @@
+hello, world!

git mv

  • Rename tracked file
  • Move tracked file from one folder to another
jopasserat@vapula:/tmp/seance3/exo1$ git mv foo pouet
jopasserat@vapula:/tmp/seance3/exo1$ mkdir dir
jopasserat@vapula:/tmp/seance3/exo1$ git mv toto dir/
jopasserat@vapula:/tmp/seance3/exo1$ git ls-files
bar
dir/toto
pouet

git rm

  • Remove file from disk
  • Stop tracking a file
jopasserat@vapula:/tmp/seance3/exo1$ git rm pouet
rm 'pouet'
jopasserat@vapula:/tmp/seance3/exo1$ ls
bar	dir
jopasserat@vapula:/tmp/seance3/exo1$ git rm --cached dir/toto
rm 'dir/toto'
jopasserat@vapula:/tmp/seance3/exo1$ ls
bar	dir
jopasserat@vapula:/tmp/seance3/exo1$ git ls-files
bar

Intermediate commands: dealing with branches

git branch

  • Create a new branch
  • List available branches
jopasserat@macbook-pro:/tmp/seance3/exo2$ git branch

jopasserat@macbook-pro:/tmp/seance3/exo2$ git ls-files

* master

  • Change to given branch
  • Can also create new branches (-b)
